Networking Project

Secure VLAN Infrastructure

A comprehensive network architecture designed using Cisco Packet Tracer, focusing on security, scalability, and logical segmentation of corporate departments.

Project Overview

This project involved designing a robust network for a multi-floor university facility. The primary goal was to ensure departmental isolation and security using VLANs, while maintaining high-speed connectivity through a multilayer switch backbone. It implements port security to prevent unauthorized access and Inter-VLAN routing for controlled communication between staff and student networks.
