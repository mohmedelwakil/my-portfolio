C# Offline Games

Logic & Arcade Collection

A diverse suite of classic arcade and puzzle games developed using C# and the .NET framework. This project demonstrates complex state management, collision detection, and user interface design in a desktop environment.

Project Overview

This collection serves as a portfolio of various game mechanics implemented from scratch. From the classic logic of "Hangman" and "Word Guess" to the real-time movement and collision handling in "Pac-Man" and "Snake," each game was built to master different programming challenges. The project also includes a "Rock Paper Scissors" simulator and a digital drawing tool, showcasing a broad range of interactive software capabilities.
